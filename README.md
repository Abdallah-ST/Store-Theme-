# PetteLuxe - Premium Pet Fashion Shopify Theme

A luxury, modern Scandinavian-inspired Shopify Online Store 2.0 theme for premium pet clothing, toys, and accessories.

## Theme Features

- **Mobile-first responsive design**
- **Sticky navigation** with search and cart
- **Hero banner** with split layout (Dogs/Cats)
- **Product filtering and sorting**
- **Advanced product gallery** with thumbnails
- **Customer testimonials section**
- **Newsletter integration**
- **Cart drawer** with upsell functionality
- **Luxury footer design**
- **SEO optimized** with schema markup
- **Core Web Vitals optimized**
- **WCAG accessibility standards**
- **Lazy loading images**

## Color Palette

- **Background**: #FAF8F4
- **Primary Text**: #1F1F1F
- **Accent Gold**: #D4AF37
- **Secondary Beige**: #EAE3D5
- **White**: #FFFFFF

## Typography

- **Headings**: Playfair Display
- **Body**: Inter

## Installation

1. Download the theme ZIP file
2. Go to Shopify Admin > Sales Channels > Online Store > Themes
3. Click "Add theme" > "Upload ZIP file"
4. Select the PetteLuxe theme ZIP
5. Click "Upload"

## Customization

All theme settings can be customized through the Shopify Theme Editor.
